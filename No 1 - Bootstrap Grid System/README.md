## sd'ler ne anlama geliyor (Burası Copy-Paste)

- .col-sm for larger mobile phones (devices with resolutions ≥ 576px);
- .col-md for tablets (≥768px);
- .col-lg for laptops (≥992px);
- .col-xl for desktops (≥1200px)
